# 🌐 CCNA  01 - Network Fundamentals

## 📘 Overview

Today, I learned the basic concepts of computer networking and the devices used in a network.

---

## 📚 Topics Covered

### 🔹 What is a Network?

A network is a group of devices connected together to share data and resources.

---

### 🔹 Computer Network

A computer network allows computers and other devices to communicate with each other.

---

### 🔹 Network vs Internet

**Network**

* A connection between devices.
* Can be small or large.

**Internet**

* A global network that connects millions of networks worldwide.

---

### 🔹 Nodes

A node is any device connected to a network.

Examples:

* Computer
* Laptop
* Printer
* Router

---

### 🔹 Hosts

A host is a device with an IP address that can send or receive data on a network.

Examples:

* PC
* Server
* Mobile Phone

---

### 🔹 Clients

Clients are devices that request services or data from a server.

Examples:

* Web Browser
* Mobile App

---

### 🔹 Servers

Servers provide services, data, or resources to clients.

Examples:

* Web Server
* Email Server
* File Server

---

### 🔹 Servers and Clients

Clients send requests.

Servers respond to those requests.

Example:

* Browser → Client
* Website Server → Server

---

### 🔹 Switches

A switch connects devices within the same network and forwards data to the correct device.

---

### 🔹 Routers

A router connects different networks and forwards traffic between them.

---

### 🔹 Firewalls

A firewall monitors and filters network traffic based on security rules.

It helps protect networks from unauthorized access.

---
