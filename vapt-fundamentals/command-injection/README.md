# 🚀 Command Injection

## 📌 Overview
This section shows my practical work on command injection.

I tested how input fields can be used to run system commands when the application does not properly check user input.

The goal was to understand how attackers use this to get access to the system.

---

## 🧪 Labs Covered
- **DVWA**  
  Tested command injection using input field and executed basic system commands.

- **PortSwigger Lab**  
  Performed simple command injection by changing input and checking output.

---

## 🧠 Key Learning
- How command injection works  
- How input fields can be abused  
- Execution of basic system commands  
- Basic idea of reverse shell  
- How attacker can access system data  

---

## 🛡️ Prevention Awareness
- Always validate user input  
- Do not trust user input directly  
- Avoid using system commands with user input  
- Use proper security checks  

---

## ⚠️ Note
This practice was done in a safe lab environment for learning purpose only.
